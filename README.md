cpp_Data_Structure
==================

Simple example for data struct: Binary Tree Traversal.

         A
        / \
      .    B
          / \
         C   D
        / \ / \
       E  F G  H
      /\ /\ /\ /\
     . .. .. .. .
    
Input: A.BCE..F..DG..H..
Output:
  Preorder(DLR): ABCEFDGH
  Inorder(LDR): AECFBGDH
  Postorder(LRD): EFCGHDBA

Leaf Count: 0
Height: 4
